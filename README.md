# 🤖 Assistente Flutuante - Lu da Magalu (Em Desenvolvimento)

Projeto em desenvolvimento de um assistente flutuante em Python.

## 🚀 Objetivo

Criar uma interface gráfica interativa capaz de:

- Executar comandos
- Abrir aplicativos
- Interpretar textos digitados
- Funcionar como assistente pessoal no Windows

## 🛠 Tecnologias utilizadas

### Atualmente:
- Python
- Tkinter
- PyQt5
- Git
- GitHub

### Planejado (próximas versões):
- Integração com Inteligência Artificial
- Processamento de Linguagem Natural (NLP)
- APIs de IA (ex: OpenAI ou chatGPT)
